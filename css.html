CSS – Cascading Style Sheets

1. O que é CSS?

CSS (Cascading Style Sheets, ou Folhas de Estilo em Cascata) é a linguagem usada para definir a aparência visual de páginas web feitas em HTML. Ele controla cores, fontes, tamanhos, espaçamentos, bordas, posicionamento e muitos outros aspectos visuais dos elementos HTML.

2. Por que o CSS é importante?

Separar o conteúdo do design é fundamental. Com CSS, podemos alterar a aparência de várias páginas de forma consistente, sem precisar editar cada página individualmente. Além disso, CSS permite criar páginas responsivas que funcionam bem em computadores, tablets e celulares.

3. Tipos de CSS

3.1 CSS Inline

Aplicado diretamente nas tags HTML usando o atributo style. Exemplo: 

<p style="color: red; font-weight: bold;">Texto vermelho e em negrito</p>

Não é recomendado para projetos grandes, pois mistura conteúdo com design.

3.2 CSS Interno

Definido dentro da própria página HTML usando a tag <style> no head. Exemplo: 

<style>
h1 {
    color: blue;
    text-align: center;
}
</style>

3.3 CSS Externo

Definido em um arquivo separado (como style.css) e vinculado ao HTML com <link>. É o método mais usado, porque permite reaproveitar estilos em várias páginas e facilita a manutenção.

4. Propriedades importantes do CSS

4.1 Cores

Podemos alterar a cor do texto e do fundo usando color e background-color. Exemplo:

  • color: vermelho → texto em vermelho
  • background-color: azul → fundo azul

4.2 Fontes e tamanhos

  • font-family → define o tipo de letra
  • font-size → define o tamanho da fonte

4.3 Margens e espaçamento

  • margin → espaço externo
  • padding → espaço interno

4.4 Bordas

Exemplo:

  • border: 2px solid black → borda preta de 2px sólida

4.5 Layout e posicionamento

Algumas propriedades importantes:

  • text-align → alinhamento do texto
  • display → block, inline, flex
  • float → posiciona elementos ao lado

7. Boas práticas

  • Separar conteúdo (HTML) de estilo (CSS)
  • Usar CSS externo sempre que possível
  • Testar em diferentes tamanhos de tela (responsividade)
  • Manter cores e fontes consistentes

8. Conclusão

O CSS é essencial para criar páginas web bonitas, funcionais e responsivas. Aprender suas propriedades, tipos, exemplos visuais e boas práticas garante um site completo e bem estruturado.
