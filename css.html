CSS – Cascading Style Sheets

1. O que é CSS?

CSS (Cascading Style Sheets – Folhas de Estilo em Cascata) é a linguagem usada para definir o estilo visual de páginas web feitas em HTML. Ele controla cores, fontes, tamanhos, espaçamentos, bordas, layouts e muito mais, tornando as páginas mais bonitas e organizadas.

2. Importância do CSS

Separar o conteúdo da apresentação é essencial. Com CSS você pode:

  • Alterar a aparência de várias páginas de forma consistente
  • Criar páginas responsivas para diferentes telas
  • Manter o código mais limpo e fácil de manter

3. Tipos de CSS

3.1 CSS Inline

Aplicado diretamente em uma tag HTML usando o atributo style. Exemplo: 

<p style="color: red; font-weight: bold;">Texto vermelho e em negrito</p>

Não recomendado para grandes projetos, pois mistura conteúdo e estilo.

3.2 CSS Interno

Definido dentro do <head> da página usando a tag <style>. Exemplo: 

<style>
h1 {
    color: blue;
    text-align: center;
}
</style>

Útil quando o estilo será usado apenas nesta página.

3.3 CSS Externo

Definido em arquivo separado (style.css) e vinculado com <link>. Exemplo: 

<link rel="stylesheet" href="style.css">

Método mais recomendado para projetos grandes, pois facilita manutenção e reutilização.

4. Propriedades importantes do CSS

4.1 Cores

Podemos alterar a cor do texto (color) e do fundo (background-color). Exemplo visual abaixo:

4.2 Fontes e tamanho

  • font-family → define tipo de letra
  • font-size → define tamanho da fonte

4.3 Margens e espaçamento

  • margin → espaço externo
  • padding → espaço interno

4.4 Bordas

  • border → define borda de elementos

4.5 Layout e posicionamento

  • text-align → alinhamento de texto
  • display → block, inline, flex
  • float → posiciona elementos lado a lado

6. Boas Práticas

  • Separar conteúdo (HTML) de estilo (CSS)
  • Usar CSS externo sempre que possível
  • Manter cores e fontes consistentes
  • Testar a página em diferentes tamanhos de tela (responsividade)

7. Conclusão

O CSS é essencial para criar páginas web bonitas, funcionais e responsivas. Aprender suas propriedades, tipos, exemplos visuais e boas práticas garante um site completo e bem estruturado.
